##### Description:

##### This project is a website about yarn. It shows you crochet projects, knitting projects, and embroidery projects, and it has tools like yarn calculator and color generator.



##### Technologies Used:

* HTML
* CSS
* JavaScript

##### How to use: 

1. open index.html in a browser
2. click toggle cozy mode to toggle dark mode
3. click the cozy yarn button in the nav bar and go to (crochet, embroidery, or knitting)
4. take a look on the cards and add the completed projects counter, then watch the youtube video and visit the channel
5. go to the tools page; you can see multiple tools
6. calculate how much yarn you need, generate a pattern, pick a color, choose your best yarn choice, and see which yarn are you, then enjoy making a yarn project

##### 

##### Features:

* Yarn Calculator
* Pattern Generator
* Color Picker
* Yarn Info
* Yarn Quiz
* info about yarn crafts and where to start learning, with a project counter
